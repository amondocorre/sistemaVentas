Instrucciones para subir el logo del sistema

- Ruta para subir el logo (desde el proyecto): `backend/uploads/logos/`
- Formatos recomendados: `svg` (preferido), `png` (512x512 recomendado), `ico` (para compatibilidad antigua)
- Nombres recomendados: `logo.svg` o `logo.png`. Si reemplazas el favicon, sube también el archivo en `assets/favicon/` o pide que se actualice ahí.
- Pasos rápidos:
  1. Sube tu archivo al directorio `backend/uploads/logos/` vía FTP o con un commit al repositorio.
  2. Si quieres usarlo como favicon, reemplaza `assets/favicon/favicon.svg` por tu `logo.svg` o avísame y lo hago yo.

Consejos:
- Mantén una versión en SVG para escalabilidad y una PNG 512x512 para apps móviles.
- Si necesitas, puedo generar los distintos tamaños de icono (16x16, 32x32, 180x180, 192x192) a partir de tu SVG.
